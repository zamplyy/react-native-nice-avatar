# react-native-nice-avatar

React Nice avatars but for React Native

## Installation

```sh
npm install react-native-nice-avatar
```

## Usage

```js
import {Avatar} from "react-native-nice-avatar";

// ...
<View>
  <Avatar>
</View>
```
